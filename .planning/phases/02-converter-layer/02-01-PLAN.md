# Phase 02: Converter Layer - PLAN 01

**Transform Pydantic models to Diode Entity protobuf messages for gRPC transmission**

## Objective

Create a converter layer that transforms the Pydantic `DiodeDevice` model into Diode SDK Entity protobuf messages. This is the translation layer that converts user-friendly Python dictionaries into the protobuf format required by the Diode gRPC API.

**Purpose:** Enable seamless conversion from validated Pydantic models to Diode SDK Entity objects for gRPC transmission to NetBox Diode. The converter must handle nested objects (interfaces, VLANs) by creating separate Entity objects rather than nesting them in the Device entity.

**Output:** Converter module with conversion functions for Device, Interface, and VLAN entities.
