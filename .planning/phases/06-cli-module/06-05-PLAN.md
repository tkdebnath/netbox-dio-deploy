## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| CLI-05-T01 | D - Denial of Service | Large batch size | mitigate | Limit max batch size to 10,000, warn on large sizes |
| CLI-05-T02 | D - Denial of Service | Infinite progress | mitigate | Add timeout for progress, limit iterations |
| CLI-05-T03 | T - Tampering | Batch size manipulation | mitigate | Validate integer range (1-10000), reject negative values |
| CLI-05-T04 | D - Denial of Service | Progress bar spam | mitigate | Limit update frequency (e.g., every 10%), throttle output |
| CLI-05-T05 | I - Information Disclosure | Progress information | mitigate | Avoid showing sensitive file paths or device details in progress |
| CLI-05-T06 | D - Denial of Service | Memory exhaustion from chunks | mitigate | Stream chunks, don't keep all in memory simultaneously |
| CLI-05-T07 | E - Elevation of Privilege | Chunk file access | mitigate | Validate chunk file paths, use temp directory for chunk storage |
</threat_model>
