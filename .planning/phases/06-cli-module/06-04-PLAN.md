| CLI-04-T03 | I - Information Disclosure | Sensitive data in output | mitigate | Mask sensitive fields (serial, asset_tag) with option, sanitize custom_fields |
| CLI-04-T04 | S - Spoofing | File extension spoofing | mitigate | Validate output file extension against allowed formats |
| CLI-04-T05 | D - Denial of Service | Disk space exhaustion | mitigate | Check available disk space, limit output size |
| CLI-04-T06 | T - Tampering | Format switching attack | mitigate | Validate format choice, sanitize output based on format |
| CLI-04-T07 | E - Elevation of Privilege | Path traversal in output file | mitigate | Validate output path, reject paths with `..`, restrict to user-writable locations |
