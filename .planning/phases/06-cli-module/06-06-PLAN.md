|----------|-------------|
| CLI -> all commands | Command routing |
| All commands -> Diode API | gRPC calls with credentials |
| User input -> All options | Input validation |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| CLI-06-T01 | D - Denial of Service | Command flooding | mitigate | Rate limit commands, implement command queue |
| CLI-06-T02 | E - Elevation of Privilege | Command chaining attack | mitigate | Validate command sequence, prevent dangerous combinations |
| CLI-06-T03 | T - Tampering | Help text manipulation | mitigate | Sanitize help text, validate all output |
| CLI-06-T04 | I - Information Disclosure | Stack traces in errors | mitigate | Only show user-friendly messages in normal mode, show full details in --debug mode |
| CLI-06-T05 | D - Denial of Service | Memory exhaustion from chaining | mitigate | Limit chain depth, process each command independently |
| CLI-06-T06 | E - Elevation of Privilege | Environment variable injection | mitigate | Sanitize environment variables, use subprocess with env parameter |
| CLI-06-T07 | T - Tampering | Exit code manipulation | mitigate | Verify exit codes in tests, don't rely on external commands for exit |
| CLI-06-T08 | D - Denial of Service | Infinite recursion in commands | mitigate | Implement command depth limit, prevent circular references |
</threat_model>
