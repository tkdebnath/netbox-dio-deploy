|----------|-------------|
| User input -> CLI parser | Command line arguments and options |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| CLI-02-T01 | T - Tampering | Argument parsing | mitigate | Use Typer's built-in type validation and validation decorators |
| CLI-02-T02 | T - Tampering | Subcommand injection | mitigate | Whitelist allowed subcommands, reject unknown commands |
| CLI-02-T03 | I - Information Disclosure | Debug output | mitigate | Only show detailed errors in --debug mode, sanitize sensitive data |
| CLI-02-T04 | D - Denial of Service | Infinite loops | mitigate | Implement command timeouts, process limits |
| CLI-02-T05 | S - Spoofing | Command spoofing | mitigate | Verify command path, validate subcommand names against whitelist |
| CLI-02-T06 | E - Elevation of Privilege | Code injection | mitigate | Use subprocess with list args only, never shell=True |
</threat_model>
